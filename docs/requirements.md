# Video Downloader - 需求分析文档

## 项目背景

用户有下载保存视频到本地的需求，但面临以下痛点：
1. 部分视频平台不支持直接下载
2. 下载不够方便（不支持批量、有限制、清晰度受限）

## 核心需求

### 必须实现 (MVP)
- [x] 支持多平台视频下载（YouTube、Bilibili、抖音、快手等）
- [x] 批量下载功能
- [x] 格式/清晰度选择
- [x] 下载进度实时显示
- [x] 手机/PC 响应式适配

### 核心技术突破

#### 1. 全平台视频无缝解析
**实现方案：**
- 利用 yt-dlp 强大的底层能力（支持 1847+ 提取器）
- 配合自定义脚本，实现对 B站、YouTube 等主流平台的视频链接解析与高速下载
- 无需用户登录，直接通过公开 API 解析

#### 2. 攻克"防盗链"机制（403/503 报错）
**问题：**
- B站等平台有 Referer 防盗链限制
- 直接访问视频资源返回 403/503 错误
- 封面图无法正常显示

**解决方案：**
- 后端增加图片代理 API (`/api/proxy/image`)
- 自动添加合适的 Referer 和 User-Agent 请求头
- 伪装浏览器请求，绕过防盗链检测
- 视频下载时也应用相同的请求头策略

#### 3. 突破免 Cookie 下载壁垒
**问题：**
- 抖音等平台必须要求用户登录态才能下载
- 传统的扫码获取 Cookie 方案存在安全风险
- 用户操作复杂，体验差

**解决方案：**
- 规避危险的扫码获取方案
- 利用 AI 挖掘网络最新的开源绕过策略
- 实现真正面向公众的"万能"免登录下载
- 通过技术手段绕过登录限制，无需用户干预

## 用户场景

### 场景1：内容创作者
> 需要批量下载参考素材，用于学习和二次创作

需求：批量输入链接、选择最高清晰度、快速下载

### 场景2：普通用户
> 看到喜欢的视频想保存到本地随时观看

需求：简单粘贴链接、一键下载、手机可用

### 场景3：知识工作者
> 下载教程、讲座视频，需要字幕和总结

需求：音频提取、字幕下载、AI 总结（V2）

## 竞品分析

| 产品 | 优点 | 缺点 |
|------|------|------|
| 各种视频平台官方下载 | 稳定 | 有限制、有水印、部分不支持 |
| 在线解析网站 | 方便 | 广告多、不稳定、不安全 |
| yt-dlp 命令行 | 功能强大 | 门槛高、普通用户不会用 |

**我们的优势：** 开源核心 + 精美界面 + 批量下载 + 后续 AI 增值功能

## 商业模式（规划中）

### 免费版
- 每日下载次数限制（如 5 次）
- 基础清晰度

### 付费版
- 无限下载
- 最高清晰度
- 批量下载优先队列
- AI 总结/字幕功能
